# Arduino Nano

## Specifications
- ATmega328 AVR
- 32 KB Flash Memory of which 2 KB used by bootloader
- SRAM 	2 KB
- Clock Speed 	16 MHz
- Analog IN Pins 	8
- EEPROM 	1 KB
- DC Current per I/O Pins 	40 mA (I/O Pins)
- Input Voltage 	7-12V
- Digital I/O Pins 	22 (6 of which are PWM)
- PWM Output 	6
- Power Consumption 	19 mA
